# List of NHS trusts
- Download the file from [here](https://www.england.nhs.uk/publication/nhs-provider-directory-and-registers-of-licensed-healthcare-providers/) and change extension to csv
- Delete all information other than trust names
- Add a `"` to all trust names; and make in to a `json` list by enclosing in `[]`
- Save file as json


# Type Notes
## Date
* Specify `default`, if used, as `yyyy-MM-dd` 
* Can also specify `today` to set to today's date
